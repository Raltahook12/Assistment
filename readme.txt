В данной программе реализованы 2 класса - Task и TaskManager. Task - класс, который представляет собой модель задачи, описывает атрибуты задачи, такие как id,название(title),описание(description), категорию задачи(category), дату выполнения(due date), приоритет(priority), и статус(status). Также в модели переопределён метод __eq__ для того, чтобы сравнивать задачу с другими задачами по значениям всех атрибутов кроме id с целью избежать дублирования задач в списке. И реализованы функции to_dict и from_dict для преобразования в словарь и последующего сохранения и наоборот, чтения из словаря.

TaskManager выполняет операции над задачами - считывает из сохранённого файла формата json(load_tasks), добавляет новые задачи(add_task), выводит выбранные задачи или категории в консоль(view_tasks), редактирует существующие задачи(edit_tasks), отмечает задачи выполненными(mark_completed), удаляет задачи(delete_task), выполняет поиск задач по названию, описанию или категории (search_task). 

Работа менеджера осуществляется через функцию main и командами через консоль интерпретатора. 
Список команд:
1 - Просмотр задач, возможен как по категориям(если указывать) или всего списка задач(если не указывать категорию)
2 - Добавление задачи. Поочерёдно введите введите название, описание, категорию, дату выполенния(в формате ДД.ММ.ГГГГ) и приоритет задачи. Менеджер автоматически присвоит задаче id и проверит задачу на дублирование в базе
3 - Редактирование задачи. Позволяет полностью изменить все значения атрибутов задачи с выбранным id.
4 - Отметить задачу выполненной. Изменяет статус задачи на "Выполнена"
5 - Удаление задачи. Удаляет задачу с выбранным id или всю категорию
6 - Поиск задачи. Выполняет поиск задачи по ключевым словам в названии или описании, категории или статусу. 
7 - Выход. Завершает работу программы
